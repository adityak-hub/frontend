Scope: scope determines the accessibility of variables,objects,functions from different parts of the code,
1.function scope.
2.Block scope.
3.lexical scope.
4.global scope.


1. function scope=variables defined inside a function are not accessible(visible) fro outside the fuunction.


ex:1---

let sum = 54; // gobal scope.   

function calsum(a,b){
    let sum= a+b; //function Scope
    console.log(sum)  --3
}

calsum(1,2);
console.log(sum);  --54

jab global scope and function scope aapas mai priority ke liye fight karte hai toh function ke andar priority ke liye function scope jitega and global ki priority ke liye global scope jitega.



2.  Block Scope = Variables declare inside a {} block cannot be accessed from outside the block,

* var pe kabhi block scope apply nhi jota. sirf let and const pe hota hai, isliye var use karna bad idea hota hai.

ex--
{
    var a = 25; 
}

console.log(a);  // print ho jaayega 


{
    let a = 25; 
}

console.log(a); // it will show an error;


for(let i=1; i<=5; i++){
    console.log(i); //block scope;
}



3.Lexical Scope=a varible defined outside a function can be accessible inside another function defined after the variable declaration.

// we use it in nested function concept.

ex----

function outerFunc(){

    function innerFunc(){   
        console.log(x);
    }
    innerFunc();

outerFunc();




..The opposite is NOT true.

***callback
A callback is a function passed as argument to be executed later.(ek function jo kisi dusre function ke argument ke form mai diya jaata hai aur baad mai call hota hai to usse callback function khte hai)
ex---
function hello(callback) {
    console.log("Hello");
    callback();
}

function bye() {
    console.log("Bye");
}

hello(bye);

Step-by-step:
hello(bye) call hua
console.log("Hello") run hua
👉 Output: Hello
callback() run hua
👉 ye actually bye() hai
bye() execute hua
👉 Output: Bye

output--
Hello
Bye


ex---
function processStudent(name, callback) {
    console.log("Processing student:", name);
    callback(name);
}

function showResult(name) {
    console.log(name + " has passed");
}

processStudent("Aditya", showResult);
